                                                                                              Rodolfo Carlos Durante krlos_19915991 • 3 months ago

                                                                                              Krlos, para que el ejemplo funcione en ese punto route.js tiene que quedar de la siguiente forma:

                                                                                              function route(handle, pathname) {
                                                                                              console.log("A punto de rutear una peticion para " + pathname);
                                                                                              if (typeof handle[pathname] === 'function') {
                                                                                              handle[pathname]();
                                                                                              }
                                                                                              }
                                                                                              exports.route = route;

                                                                                              El if es para que la aplicación no se cuelgue cuando el browser intenta cargar el favicon. Igualmente si continuás con el tutorial el código de los ejemplos se reacomoda y se pueden comprender bien todos los conceptos.
